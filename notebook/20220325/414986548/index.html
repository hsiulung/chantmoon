    [Hexo] Themes Stellar

    發表於 2022-03-25 更新於 2022-05-14 分類於 技術 閱讀次數： 所需閱讀時間 ≈ 2 分鐘

    執行

    • hexo s / 預覽
      • http://localhost:4000/
      • http://localhost:4000/admin/
    • hexo g / 重新產生
    • hexo d / 發行

    hexo clean && hexo g && hexo s
    hexo clean && hexo g && hexo d

    Stellar

    • https://xaoxuu.com/wiki/stellar/wiki-settings/#%E9%A1%B9%E7%9B%AE%E7%9A%84-GitHub-%E4%BB%93%E5%BA%93%E4%BF%A1%E6%81%AF

    安裝

    1. Hexo
      install
      https://nodejs.org/en/

    2. init
      npm install -g hexo-cli
      npm install //安裝 Hexo
      npm install hexo-deployer-git –save //安裝 git 部署套件

    3. setting
      git config –global user.email “hsiulung417@hotmail.com“
      git config –global user.name “hsiulung”
