休養生息

發表於 2022-07-21 分類於 日常 閱讀次數： 所需閱讀時間 ≈ 4 分鐘

2022-07-16

https://www.plurk.com/p/ox9mw7
之前發了對於創作的噗，算是對未來的決策？

我最近在想，把藍月想改的地方改一改之後我應該會很少寫東西 。

畢竟寫東西也寫了不少，整體來說……

算是很失敗的時間投資。

不過完全不寫或者畫東西又真的不知道幹嘛，總覺得人生還是要培養個除了玩小遊戲跟看漫畫以外的興趣，否則真的很空虛。

這次大概又不會把安息者貼完就會把小帳號關了，總覺得這個故事某種程度上真的是……

……

唉，覺得非常浪費時間，每次貼都覺得自己很可悲，最後還真的是有個人點了訂閱所以我才很勉強的維持低程度的更新，否則我大概一個字都不會想寫。

我後來在想，我還是喜歡偶爾寫東西，但是我會考慮永遠都不要貼也不去發類似的東西，因為純粹是在浪費時間加上消耗動力 。

雖說我的時間也沒有很貴，但這樣自我消耗下去完全不是辦法 。

總不能每次都在做重複的事情，又抱怨同樣的痛苦，我下定決心這次要認真考慮。

我覺得為此感到沮喪很煩，卻也對此毫無辦法，或許人就是喜歡做夢喜歡飛蛾撲火。

但即使如此也要畫個界線，遵守自己定下的規則比較好。

分析覺得麻煩的點

  1. 貼文很麻煩，寫不會
  2. 以此交友變成條件交換的回應也很麻煩，對此感到沮喪也很浪費情感
  3. 1+2通常講什麼都會變成自言自語，非常麻煩
  4. 對於為此騷動的自己跟所有人都感到很厭煩
  5. 實際上想講的東西很多都不該透漏，但是如果刪掉這些這個帳號會都是玩遊戲跟看漫畫心得

等到恢復狀態的時候我又會想要打開，所以我常常搞不清楚我到底想要什麼 。

實際上對我來說最佳狀態是寫而不發更不討論，因為我其實已經放棄寫好這件事。

我其實對於別人怎麼看自己的文已經沒有興趣，跟風之類也不會想寫，同時也對類似的文手互動興致超級低，這樣就……就變成一個遮遮掩掩的日常噗，我連這帳號為什麼還要留著都很迷惘。

因為我很不喜歡更藍月，接下來也不會有其他文，我還在掙扎想把文帳處理掉，這次應該是乾脆把友粉都挪掉直接放置當成遺跡，反正做不到的就是做不到乾脆承認比較快樂。

現在掙扎的地方在應該

  1. 發公告後動手
  2. 直接動手
  3. 把密碼改到自己都不知道變成廢墟
  4. 刪掉所有噗打開帳號
    反正煩惱一陣子最後我應該選擇2

這個帳號我之前也考慮刪，理由同上，做不到的事情就是真的做不到，而且他好像也十年是時候可以處理了，帶不走的東西就放在這裡會不會比較好。

雖然有認識的朋友，但是……好像……
差不多緣分已盡。

我已經不是以前的我，更不能想像這個帳號的未來，因為他沒有未來。

我從去年就開始思考要不要刪帳，可能會考慮到今年九月。

等到我生日那天若我還是沒改變主意應該會做跟文的帳號一樣的事情。

我還在糾結這件事，然後我想到要是以後我再貼文、發類似的噗就要捐/花一萬元買東西給人，這光是想到就覺得很痛苦
類似的東西我要改寫在不公開的日記這樣應該是我想要的結果。

給自己的禁止事項

限制只在噗浪，反正我也不用FB
遵守關你屁事跟關我屁事的最高快樂指導原則

  1. 公開發同類的主題噗
  2. 發、回任何創作跟風噗
  3. 貼任何小說出來，但是放在自家網站可以
  4. 完稿前聊創作心得跟任何改稿過程
  5. 任何嘗試得到創作回應的舉動
  6. 回應任何創作、委託、轉蛋的噗
  • 懲罰項目：捐一萬，標的是之前看到的殯葬相關
  • 時間：改完藍月之前 或者永久

明天睡醒把限制寫出來列印超大字貼出來，我真的不要在這種事情上面浪費生命了……
每次想發，就要提醒自己珍視沒多少的錢跟時間，想要點開討論也絕對不能點因為發噗超貴

2022-07-22

實際測試後發現回應可以比較自由，因為對我影響並不高。

因為我完全可以選擇不看且不回應。

不會有人逼著我表達看法，所以我當然也可以視若無睹，這是我可以選擇的？

最近透露出想走的想法後，被逼著一直丟職缺資訊順便要我好自為之，我看下來整個很不爽。

所以我開始思考為什麼。

抱怨的事情我也不想重複抱怨，但我現在已經一個月來兩次月經，展現出的就是我精神跟健康狀態確實不好，已經實際影響到我的身體。google後可能要繼續觀察，畢竟我現在狀況符合壓力大+作息不正常，前者我只要斷掉跟同事的下班訊息可以減少一半，剩下的靠自己作息。

0720 memo

  1. 我為何感到煩躁？
  2. 我為何在跟同事的對話中感到壓力？
  3. 我能夠如何處理？
  4. 我應該如何劃下界限？
  5. 我想要反擊嗎？如果反擊，想要達到什麼目的？

思考之後我開始做出些許改變，如果這些改變能夠降低我的痛苦。

並不是不發牢騷就不會痛苦，也不是會回應就表示期待並想要看到這訊息……

我自己也做過丟垃圾的人，當時我被罵一直牢騷的時候我選擇直接不跟對方講任何的話，因為你一直責怪我牢騷，但你沒有透漏出任何訊息表示你現在很忙或者不想聽，這讓我耿耿於懷。

一方面我會懂對方希望我自己讀空氣的想法，另一方面也知道這不可能。

所以我這次立場交換，我選擇實話實說，做出可以實際改善我心情的作為而不是直接絕交處理。

  1. 禮貌修飾但直接表達，我無意對下班不斷聽別人的工作細節牢騷。
  2. 表達我要不要走關你屁事，我自己會考慮，不要繼續逼我也不要用「你好自為之」的態度跟我講話。
  3. 我不喜歡聽不熟的人所有無法證實的八卦推測，我也不在意要走的人為何走。←可惜這我講了七八次了，對方自己不避嫌而被人八卦很不爽，但他其實本質也很八婆。
  4. 比起情感導向的推測，我比較想知道確定的未來應該如何面對。
  5. 下班後手動消音對方，直接不回。
  6. 我有大腦，我也不見得比你愚蠢，我的未來我會自己承擔，與你無關。不要想控制我。

時間太久，我已經不太記得當時的事件。

但我的想法是，並不是非要丟出垃圾要逼人接收我的情緒垃圾不可，但我希望你要暴怒之前起碼透露出你很忙或者不想聽的明確訊號。

1-4項我有很明確地講過了，5已經執行中，但是6這個我還在思考，我要更多訊號確認他是否真是在用不好察覺的方式試圖控制我的決策，進一步判斷以及他想控制我的動機是什麼。

是為了他自己的利益？是認為他的判斷比我更優越？

無論如何，我不想重複同樣的錯誤。

該劃界限的時候我會劃的。
